## Execution - dev mode

To execute the app you need to use two consoles, in first console you should execute:


```sh
sh backend.sh
```

And the second: 
```sh
sh frontend.sh
```

Subsequently go to your browser and access to [localhost:3000](http://localhost:3000) for see the app running! :)
